# Project Plan: Apple Financial Analysis & 3-Year Forecast

## Phase 1: Data Collection
- Collect Apple financial statement data for the last 5 fiscal years.
- Gather revenue, gross profit, operating income, net income, assets, liabilities, equity, operating cash flow, capital expenditures, and free cash flow.

## Phase 2: Data Cleaning
- Organize financial data by fiscal year.
- Standardize account names.
- Calculate year-over-year growth rates.
- Prepare a clean dataset for Excel and dashboard use.

## Phase 3: Financial Analysis
- Analyze revenue growth.
- Analyze gross margin, operating margin, and net margin.
- Analyze liquidity and balance sheet strength.
- Analyze operating cash flow and free cash flow.

## Phase 4: Forecasting
- Build a 3-year forecast.
- Create base case, upside case, and downside case.
- Forecast revenue, net income, and free cash flow.

## Phase 5: Dashboard
- Build an executive summary dashboard.
- Create a profitability dashboard.
- Create a balance sheet and cash flow dashboard.
- Create a forecast dashboard.

## Phase 6: Final Report
- Write executive summary.
- Document key findings.
- Provide business recommendations.
