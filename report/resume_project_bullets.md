# Resume Bullets: Apple Financial Analysis & Forecast Project

## Project Title
Apple Financial Analysis & 3-Year Forecast

## Resume Bullets

- Built an end-to-end financial analysis project using public Apple financial statement data to evaluate revenue growth, profitability, liquidity, leverage, cash flow performance, and shareholder value indicators.

- Developed a 5-year historical financial dataset and calculated key financial ratios, including gross margin, operating margin, net margin, current ratio, debt-to-equity ratio, return on assets, return on equity, and free cash flow margin.

- Created a 3-year scenario-based forecast model with base, upside, and downside cases to project revenue, gross profit, operating income, net income, capital expenditures, and free cash flow.

- Built Python scripts to automate financial ratio calculations, generate forecast outputs, and create visual trend analysis for revenue, profitability, balance sheet, and cash flow performance.

- Designed an executive-level dashboard framework for Power BI/Tableau to present historical performance, KPI trends, financial health, and forecast scenario analysis for business decision-making.

- Published the complete project on GitHub, including cleaned datasets, Python scripts, Excel financial model, visualizations, source documentation, and executive summary report.
